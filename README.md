🎮 Emoji Match - Jogo da Memória

Um jogo da memória moderno, responsivo e intuitivo desenvolvido com foco em performance e experiência do usuário (UX). O projeto utiliza o teclado de emojis padrão para criar um desafio visual divertido.

🚀 Demonstração

O projeto consiste em uma grade interativa onde o jogador deve encontrar pares de emojis idênticos no menor tempo e número de movimentos possíveis.

✨ Funcionalidades

Mecânica de Jogo Fluida: Sistema de virada de cartas com animações 3D realistas.

Contadores em Tempo Real: Monitoramento de movimentos e cronômetro de precisão.

Responsividade Total: Interface adaptável para dispositivos móveis, tablets e desktops.

Acessibilidade: Suporte para navegação via teclado (Tab e Enter/Space) e atributos ARIA.

Feedback Visual: Animações de sucesso (pulse) e estados visuais claros para cartas viradas ou combinadas.

Modal de Vitória: Resumo de desempenho exibido ao finalizar o jogo com opção de reinício rápido.

🛠️ Tecnologias Utilizadas

HTML5: Estrutura semântica e acessível.

Tailwind CSS: Estilização moderna através de classes utilitárias e design system consistente.

JavaScript (Vanilla): Lógica de jogo pura, manipulação do DOM e gerenciamento de estados.

Animações CSS: Transformações 3D e keyframes personalizados para transições suaves.

📋 Como Executar

Não é necessária nenhuma instalação ou compilação. Basta seguir os passos abaixo:

Baixe o arquivo index.html.

Abra o arquivo em qualquer navegador web moderno (Chrome, Firefox, Edge, Safari).

Divirta-se!

🧩 Estrutura do Código

O código foi organizado em um único arquivo para facilitar a portabilidade:

Estilos (<style>): Definições de perspectiva 3D e animações de feedback.

Estrutura (<body>): Layout dividido em Header, Painel de Status, Tabuleiro (Grid) e Modais.

Lógica (<script>):

shuffle(): Algoritmo Fisher-Yates para embaralhamento das cartas.

flipCard(): Gerenciamento de eventos de clique e lógica de comparação.

startTimer(): Controle de tempo de jogo.
